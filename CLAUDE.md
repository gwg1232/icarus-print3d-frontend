# Claude Code 규칙

## 프로젝트 규칙

### 프론트엔드
- `HTML` 대신 `HTMX`만 사용
- `CSS` 대신 `Tailwind CSS`만 사용
- `JavaScript`는 사용하지 않음
- Tailwind CSS CDN 버전 사용 (빌드 프로세스 없음)
- 데스크탑 레이아웃만 지원
- 반응형 UI 미지원
- 서버 사이드 렌더링이나 빌드 프로세스 없이 직접 브라우저에서 실행 가능

### 백엔드
- 백엔드 구현 요구사항을 `README.md`에 작성하면 별도의 백엔드 프로젝트에서 외부 API로 구현할 예정
