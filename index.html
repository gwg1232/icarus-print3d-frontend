기술 원리

탄소섬유 나일론(Onyx) 내부에 연속섬유 경로를 형성해 강한 구조를 생성합니다. 하중이 집중되는 구간에 연속섬유를 보강하여 크게 향상된 강도와 강성을 제공합니다.

일반 3D 프린팅의 한계 보완

적층 특성상 낮은 강도와 방향성 문제를 연속섬유로 해결

효율적인 구조 설계

필요한 부분에만 섬유를 보강하여 무게 대비 최고의 강도 구현

정밀 부품 생산 특화

기본 소재만으로도 강하고 튼튼하여 정밀한 부품 생산에 최적화

기존 FDM vs 연속섬유강화

❌ 기존 FDM 방식

  • • 적층 특성상 상대적으로 낮은 강도
  • • 민감한 방향성으로 인한 제약
  • • 하중 지지 능력 한계

✅ 연속섬유강화 CFF

  • • 하중 집중 구간에 연속섬유 보강
  • • 크게 향상된 강도와 강성
  • • 방향성 문제 해결
  • • 알루미늄급 성능 달성
